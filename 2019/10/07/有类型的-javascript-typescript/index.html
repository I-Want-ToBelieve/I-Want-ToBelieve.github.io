  1. 第五步是学以致用
    TypeScript 不是银弹, 需要考虑到引入 TypeScript 带来的多余复杂度(多了一个类型维度的编码),
    一般来说, TypeScript 在现阶段适合在需要长时间维护的中大型项目(或开源库)中使用,
    如果没有这种项目可以实践, 尤大不久前发布的 Vue 3 Pre-Alpha 版本的源码可以读读看.
